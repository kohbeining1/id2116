Trying out Neopixels on microbit

To test out possibilities for our 'New Switch' project



New Switch Exploration: Response for tripping



LED to change microbit is shaked or rotated. These actions to the microbit mimics common actions of the shoe when it is worn.
This helps us move forward in the idea to create an alarming media response if person wearing the shoe trips over.



New Switch Exploration: Response for jumping



LED to change color when microbit falls from a height and a certain level of acceleration. This is to mimic the jump action when shoe is worn.



Microbit Code for Neopixel response





Neopixel response is shown through a combination of trigger: using buttons on the microbit and
'freefall'/'shake'/'screen down' gestures at the same time with 'sensorval' function.





Exploration success! One step forward for our New Switch Project :)
